## Section 1: Introduction to HTML
### What is an Element?
* tag that wraps text and lets the browser know what to do with it
```HTML
<p class="foo">This is a paragraph.</p>
```
* **/<p/>** - start tag
* **/</p/>** - end tag
* **class** - attribute name
* **"foo"** - attribute value
* **This is a paragraph.** - content
